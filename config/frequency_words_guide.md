# 关键词分类说明

本文档说明 `frequency_words.txt` 中的主题分类和关键词配置。

## 文件格式

- **空行分隔** = 不同主题，每个空行创建一个新的主题分组
- **同一主题的词连续排列**，中间不能有空行

### 特殊语法

| 语法 | 说明 | 示例 |
|------|------|------|
| `#名称` | 自定义主题名称（必须作为词组第一行） | `#AI大模型` |
| 普通词 | 标题包含任意一个词即匹配 | `华为` |
| `+词` | 必须词，所有 `+` 词都要匹配 | `+AI +手机` |
| `!词` | 过滤词，匹配则排除该新闻 | `!广告` |
| `@数字` | 该主题最多显示的条数 | `@10` |
| `[GLOBAL_FILTER]` | 全局过滤区，匹配的新闻全部排除 | |

**注意**：如果不指定 `#名称`，将使用第一个关键词作为主题名称。

## 如何添加新主题

1. 在文件末尾（`[GLOBAL_FILTER]` 之前）添加空行
2. 写入新主题的关键词，每行一个
3. 相关词放在一起，不要有空行

示例：
```
新主题关键词1
新主题关键词2
新主题关键词3
```

## 如何添加过滤词

- 主题内过滤：在主题内添加 `!词`
- 全局过滤：在 `[GLOBAL_FILTER]` 区域添加
